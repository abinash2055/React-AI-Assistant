# Shifra - AI Voice Assistant

Shifra is an **interactive AI voice assistant** built with **React.js** that can **listen to your voice commands** and **speak responses** back. It leverages the **Web Speech API** for speech recognition and speech synthesis, providing a real-time, hands-free experience.

---

## 🚀 Features

- 🎤 **Voice Input**: Speak to the assistant and get real-time feedback.
- 🗣️ **Speech Output**: Assistant replies using text-to-speech.
- 🤖 **Interactive UI**: Simple and friendly user interface with animated avatar.
- 🌐 **Multilingual Support**: Supports Hindi (`hi-IN`) and English (`en-US`, `en-GB`).
- ⚡ **Lightweight**: Built purely with React and browser APIs (no heavy backend required).

---

## 🛠️ Technologies Used

- **React.js** – Frontend framework
- **Google Gemini API** – Voice recognition and speech synthesis
- **React Icons** – Interactive icons for UI
- **CSS** – Custom styling for responsive design
